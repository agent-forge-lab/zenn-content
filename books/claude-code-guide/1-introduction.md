# はじめに

## Claude Code とは何か

Claude Code は、Anthropic が提供するターミナルベースの AI アシスタントです。エディタの拡張機能ではなく、ターミナルで直接動作するエージェント型のツールとして設計されています。

従来の AI コーディングアシスタントが「コードの補完」に留まっていたのに対し、Claude Code はプロジェクト全体を理解し、ファイルの読み書き、コマンドの実行、Git 操作、テストの実行まで自律的に行います。人間の開発者と同じように、ターミナルを通じてプロジェクトに関わるのです。

```bash
# Claude Code はターミナルで起動する
claude

# プロジェクトのルートで実行すると、コードベース全体を把握する
cd my-project && claude
```

## このガイドの目的

本書は、Claude Code を日常の開発ワークフローに組み込むための実践的なガイドです。インストールから始まり、基本的な使い方、カスタマイズ、チーム開発での活用、そして応用テクニックまでを段階的に解説します。

「AI にコードを書かせる」のではなく、「AI と一緒にコードを書く」ためのノウハウを体系的にまとめています。

## このガイドの特徴

本書には一つ、ユニークな特徴があります。この本自体が、Claude Code のマルチエージェント機能を使って執筆されています。

Agent Forge チームでは、複数の AI エージェントがそれぞれの役割を持ち、協調しながらプロダクトを開発しています。本書の内容は、AI エージェントが実際にソフトウェア開発を行う中で得た知見に基づいています。

つまり、本書で解説するテクニックは机上の空論ではなく、実際に動くプロダクトの開発を通じて検証された方法論です。

## 本書の対象読者

- Claude Code をこれから使い始めるエンジニア
- AI アシスタントを開発に取り入れたいが、効果的な使い方がわからない方
- チーム開発で Claude Code を活用したい方
- エディタの AI 拡張に限界を感じている方
